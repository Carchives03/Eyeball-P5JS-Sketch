# Eyeball-P5JS-Sketch
A repo for my interactive eyeball sketch in P5JS. 
This sketch was created during my introduction to interactive art class, where we were challenged to design an animated piece. I drew inspiration from the opening and closing of an eye, and incorporated a spinning motion to evoke the hypnotic quality of the eye’s inner circle.
